一键使用（Windows 便携版）

1. 下载并解压 SubtitleMaker-Windows-Portable.zip
2. 双击运行 `字幕生成工具.exe`
3. 首次运行若提示缺少模型或依赖：
   - 点击 “GitHub下载” 或 “hf-mirror下载”
   - 或点击 “手动导入” 打开模型目录，将 tokens.txt / model.onnx / silero_vad.onnx 放入
4. 选择媒体文件或输入文件夹，点击 “开始”
5. 在输出目录 `输入文件的文件夹` 查看生成的 srt

提示
- 便携包内自带 ffmpeg，不需要安装 Python/ffmpeg
- 若失败请查看 `logs/run.log`，错误提示会说明缺少的文件路径
